Sample Usage

See the documentation for more comprehensive usage examples.

Bidirectional Maps

const map = new HashBiMap<string, number>()
 .set('foo', 3)
 .set('bar', 2);

map.get('foo'); // 3
map.hasValue(2); // true

// get a reference to the inverted map (number to string)
const inverse = map.inverse();
inverse.get(3); // "foo"
inverse.delete(2); // true

Multisets

const set = new MapMultiset<string>()
 .add('foo')
 .add('foo');

set.count('foo'); // 2

Exported Classes

Collections

  • ArrayMultimap - a multimap backed by arrays for its values
  • MultisetMultimap - a multimap backed by multisets (bags) for its values
  • HashBiMap - a bidirectional map backed by two hash tables
  • MapMultiset - a multiset (bag) backed by a map of items to counts
  • ImmutableMap - a map which is immutable (the entries cannot be changed)
