Class HashBiMap<K, V>

A BiMap backed by two hash tables. This implementation guarantees insertion-based iteration order of its keys.

Example usage

const map = new HashBiMap<string, number>()
 .set('foo', 3)
 .set('bar', 2);

map.get('foo'); // 3
map.hasValue(2); // true

// get a reference to the inverted map (number to string)
const inverse = map.inverse();
inverse.get(3); // "foo"
inverse.delete(2); // true

map.get('bar'); // undefined
inverse.get(2); // undefined
