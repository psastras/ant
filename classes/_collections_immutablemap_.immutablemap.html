Class ImmutableMap<K, V>

A Map which is immutable after construction. An {@link Error} will be thrown if an attempt is made to modify this map.

Example usage

const map = new ImmutableMap<string, number>(
  ['foo', 3], ['bar', 2]
);

map.get('foo'); // 3
map.has('bar'); // true

// the map cannot be modified
map.set('voo', 3); // throw new Error
map.clear(); // throw new Error
