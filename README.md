## Development

Install dependencies

```shell
yarn install
```

Running tests

```shell
yarn test
```

Building the distribution

```shell
jspm install
npm run build
```

When developing, it can be helpful to watch the source and tests for changes, rerunning
tests when necessary.

```shell
npm run watch
```
